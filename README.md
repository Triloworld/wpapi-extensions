Installation
------------

WP-API Extensions are available via [yarn] and [npm].

    $ yarn add wpapi-extensions
    $ npm install --save wpapi-extensions

[yarn]: https://yarn.pm/wpapi-extensions
[npm]: https://www.npmjs.org/package/wpapi-extensions
